# Car Price Scraper

Car Price Scraper is a Python application that collects vehicle prices from multiple online marketplaces and presents them in a unified format.
The project demonstrates web scraping, data processing, and data analysis using Python.

## Features

- Collect vehicle prices from multiple websites
- Process and analyze collected data
- Compare prices across different sources

## Technologies

- Python
- BeautifulSoup
- Pandas

## Installation

1. Download the rar file
2. Extract the rar file
3. Run the exe file
4. Enjoy!
